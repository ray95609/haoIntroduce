## Context

當前網站已具備基礎的劇場場景切換與導讀光點系統。然而，內容多為佔位符，且部分視覺（如 QR Code）與整體劇場氛圍不夠融合。傳單內容的加入提供了具體的設計語意（如「安全性」、「科學化」）。

## Goals / Non-Goals

**Goals:**
- 將 Scene 1 的文案拆分為「否定」與「肯定」兩個階段，提升敘事深度。
- 在 Scene 2 的項目卡片中嵌入微小的「專業標籤」，強化科學感。
- 為 QR Code 容器增加物理厚度、玻璃擬態（Glassmorphism）與光影邊框。
- 導讀光點在進入 QR 區塊時，需呈現精確的物理嵌入動畫。

**Non-Goals:**
- 不涉及後端資料庫整合。
- 不改變現有的導讀點核心移動演算法。

## Decisions

- **Slogan 呈現方式**：採用「分段不透明度動畫」。當 Scene 1 啟動時，先亮起「不是更努力」，隨後光點快速掃過時觸發「文字反轉」。這比直接顯示更有戲劇張力。
- **專業標籤實作**：使用 CSS 偽元素 (`::before`/`::after`) 或額外的 `span` 標籤，並設定極低的不透明度（如 0.3），模仿儀表板上的背景參數。
- **QR 卡片質感**：使用 `box-shadow` 的多層疊加模仿實體邊緣高光。背景使用 `backdrop-filter: blur(10px)` 增加玻璃厚度感，與背景的點陣（Dot Grid）產生視覺疊加。

## Risks / Trade-offs

- [Risk] 文字拆分過細可能導致在極小螢幕上排版跑掉 → [Mitigation] 使用 `clamp()` 與 `flex-wrap` 確保響應式。
- [Risk] 背景 Blur 效果在舊型行動裝置可能影響效能 → [Mitigation] 使用 CSS 特性檢測，必要時回退至半透明背景。
